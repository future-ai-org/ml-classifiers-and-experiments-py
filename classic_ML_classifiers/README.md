# classic ML classifiers (stony brook, 2013)

<br>

> small, self-contained Python experiments implementations of **classic machine learning** methods: boosting, generative models, discriminative models, instance-based learning, and simple sequence models. Each subdirectory is its own mini-project with scripts, data, and (where present) local notes.

<br>

| Directory | What it covers |
|-----------|----------------|
| **adaboost** | AdaBoost with decision stumps as weak learners; includes BUPA liver-disorder splits and related data. |
| **expectation_maximization** | Expectation–maximization (EM) for mixture models, plotting, and course-style data (`hw5.data`, `conf.json`). |
| **hidden_markov_model** | A first HMM example (`first_example/`). |
| **k-NN** | *k*-nearest neighbors, cosine distance utilities, and cross-validation experiments over *k*. |
| **naive_bayes_vs_logistic_regression** | Naive Bayes vs logistic regression on a sparse 20 Newsgroups–style document classification setup (`train.data` / `test.data` and labels). |
